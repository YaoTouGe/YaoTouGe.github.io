### 纹理光源的专属属性
| 属性名称               | 描述                                                                                                  |
| ---------------------- | ----------------------------------------------------------------------------------------------------- |
| IsRealTime（实时更新） | 若设为 true，纹理预过滤过程会每帧执行，会增加额外性能消耗。建议仅对视频纹理等持续变化的内容启用此选项 |
| Texture（纹理）        | 光源使用的纹理，可选择静态 Texture2D 或运行时生成的 RenderTexture                                     |
| Width（宽度）          | 纹理光源的宽度                                                                                        |
| Height（高度）         | 纹理光源的高度                                                                                        |
