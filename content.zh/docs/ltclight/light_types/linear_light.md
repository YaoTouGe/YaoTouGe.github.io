### 线性光源的专属属性
| 属性名称            | 描述                                                                               |
| ------------------- | ---------------------------------------------------------------------------------- |
| Range（范围）       | 胶囊体的范围，定义胶囊两端半球的半径和中间圆柱的长度，胶囊体外部区域不受该光源影响 |
| Start Point（起点） | 线性光源的起点，即胶囊体一端半球的球心                                             |
| End Point（终点）   | 线性光源的终点，即胶囊体另一端半球的球心                                           |
| Radius（半径）      | 与 **Range（范围）** 无关，仅表示线性光源的“粗细”，数值越大，光源看起来越亮        |
